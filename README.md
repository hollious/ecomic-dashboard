## 파일 구조

```
Test/
├── index.html    # 메인 HTML 파일
├── styles.css    # 스타일시트
└── script.js     # JavaScript 파일
```

## 실행 방법

브라우저에서 `index.html` 파일을 직접 열거나, 로컬 서버를 실행하세요.

### 로컬 서버 실행 (Python)

```bash
cd ~/Downloads/"Claude Code Test"/Test
python3 -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000` 접속

### 로컬 서버 실행 (Node.js)

```bash
cd ~/Downloads/"Claude Code Test"/Test
npx serve
```
